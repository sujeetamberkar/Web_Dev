- 1) You can select which set of HTML elements you want to apply for CSS
- 2) Universal Selector *
```
*{
    background-color: red;
}
```
- 3) Element Selector 
    h1{
    color: white;
}


- 4) ID should be unique 

```
    #id1{
    }

```

- 5) Selector List 

```
    .class1 , .class2 , .class3 {
    YOU CAN ENTER 
    }
```
